 The Next Step : Upcoming Component

The upcoming product will build for WordPress Woocommerce with the name of “SmartWooCatalog”. This component will use some “SmartListings” components as base tools like SmartUrls, SmartCatalog and SmartFilters. This component will provide the full Drag/Drop environment to build dynamic catalog with ease. That is build for both developer or non-developers. This will come with extensive number of default layout templates and themes. Developers would be able to add new control to extend its functionality. And as well designer could be easily build new themes and layouts of this product.

This component will very helpful to change product layout and adding sidebars for catalog page using Interactive (Drag/Drop) environment.

 Theming Frameworks

Due to strong background with Web UI's I always want to work on layout blocks. I have created bridge extensions for different CMS and frameworks to handling layouts with ease. For example, I have created LayoutBundle for symfony 2.5. This Bundle helps symfony users to build themes. This bundle provides the facility to create multiple themes for single web application. LayoutBundle used simple XML configuration files to manage themes quickly. In the future, I will work on an interactive Drag/Drop layout system which will help users to build customize-able layouts for different CMS and frameworks(e.i, Symfony, Laravel, WordPress, Drupal, Django).
