Any sub component could be used as stand-alone tool or work together.

SmartFilters

SmartFilters mostly work with “SmartCatalog” component, but that could be used as standalone. SmartFilters comes with two types of controls. First type of controls is “standalone control”, which could be placed inside any block or widget. For example, CheckboxGroup, this control have independent click/change events. On every option checked or unchecked the page will send notify to “SmartCatalog” for filter products. Second type of control is “a form”, which have one or many 'form fields' and submit button. This form could be submitted using clicked on submit button for send notify to “SmartCatalog” for filter products. SmartFilters controls come with multiple themes and variations(types). For example, CheckboxGroup have three variations, SimpleCheckboxGroup(for less number of options), ScrollingCheckboxGroup, PopupCheckboxGroup(for more number of options). User could add controls for SmartFilters easily.
Features

  • Build-in Controls with multiple variations
  • Sidebars
  • Widgets
  • Parent/Child

SmartCatalog

SmartCatalog is very useful component to build dynamic product catalog or item listings with ease. SmartCatalog comes with build in many useful controls and customize able featured according to users need. There are many catalog templates available for products catalog or item listings, which are easily modified able for designer or developers. User could build custom catalog or listing template or control for SmartCatalog.
Features

  • Traditional catalog(On Page Refresh)
  • Ajax catalog
  • Express catalog
  • Infinite catalog(without paging)
  • Multi mode catalog(Multiple layout like tile,listings etc)
  • Multi Regions for Listings(like featured items, regular items etc)
Controls
  • Sortbars(Multiple variations)
  • Paging (Multiple variations)
  • Dynamic listings
  • Suggested Links
  • Loading bars(Multipl variations)
